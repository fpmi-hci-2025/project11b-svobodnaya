В данной папке содержатся все схемы и материалы по лабораторной работе №2, посвященной проектированию информационной архитектуры и взаимодействия с пользователем.

## Структура

### Корневая папка
- **site_map.drawio** - Карта сайта (общая структура и навигация)

### Концептуальные макеты
Папка `layouts` содержит концептуальные макеты для двух платформ:

- эскиз главная страница с меню.drawio
- эскиз главная страница.drawio
- эскиз задача.drawio
- эскиз канбан проекта.drawio
- эскиз мастер создания задач.drawio
- эскиз мастер создания команды.drawio
- эскиз мастер создания пользователя.drawio
- эскиз мастер создания проектов.drawio
- эскиз панель админа.drawio
- эскиз панель аналитики.drawio
- эскиз пользователь.drawio
- эскиз проект.drawio
- эскиз список досок.drawio
- эскиз список задач.drawio

#### Мобильная версия (`mobile/`)
- эскиз главная страница с меню.drawio

- эскиз задача.drawio
