# Диаграммы деятельности (Activity diagrams) ProjectFlow

---

В данном подразделе представлены ключевые диаграммы деятельности:

- **diagram_activity_project_creation.puml**: процесс создания и планирования нового проекта
![diagram_activity_project_creation](img/diagram_activity_project_creation.png)
- **diagram_activity_sprint.puml**: жизненный цикл спринта/выполнения задач
![diagram_activity_sprint](img/diagram_activity_sprint.png)
- **diagram_activity_analytics.puml**: цикл аналитики и мониторинга по проекту
![diagram_activity_analytics](img/diagram_activity_analytics.png)

Каждая .puml-диаграмма снабжена детализацией этапов, пояснениями на русском языке и может быть конвертирована в картинку с помощью PlantUML.
