# Карточки для обратной сортировки
## ProjectFlow - 30 тестовых карточек

## Карточки (объекты для размещения)

### Группа 1: Задачи и управление задачами

**Карточка 1:** "Список всех задач, назначенных на меня в текущем спринте"  
**Правильный путь:** Задачи → Мои задачи → Назначенные мне

**Карточка 2:** "Kanban доска с колонками To Do, In Progress, Done"  
**Правильный путь:** Задачи → Kanban доска

**Карточка 3:** "Создать новую задачу типа Bug с приоритетом High"  
**Правильный путь:** Задачи → Создать задачу

**Карточка 4:** "Календарь с дедлайнами задач на текущий месяц"  
**Правильный путь:** Задачи → Календарь задач

**Карточка 5:** "Задачи, которые я создал, но назначил на других"  
**Правильный путь:** Задачи → Мои задачи → Созданные мной

---

### Группа 2: Проекты

**Карточка 6:** "Список всех активных проектов компании"  
**Правильный путь:** Проекты → Все проекты

**Карточка 7:** "Создать новый проект для разработки мобильного приложения"  
**Правильный путь:** Проекты → Создать проект

**Карточка 8:** "Проекты, которые я добавил в избранное для быстрого доступа"  
**Правильный путь:** Проекты → Избранные

**Карточка 9:** "Завершенные проекты за последний год"  
**Правильный путь:** Проекты → Архив

**Карточка 10:** "Проекты, в которых я являюсь участником команды"  
**Правильный путь:** Проекты → Мои проекты

### Группа 3: Команда и пользователи

**Правильный путь:** Команда → Участники → По проектам

**Карточка 12:** "Пригласить нового разработчика в команду"  
**Правильный путь:** Команда → Пригласить участника

**Карточка 13:** "Посмотреть права доступа роли Project Manager"  
**Правильный путь:** Команда → Роли и права

**Карточка 14:** "Список всех QA Engineers в компании"  
**Правильный путь:** Команда → Участники → По ролям

**Карточка 15:** "Создать новую команду для Frontend разработки"  
**Правильный путь:** Команда → Команды

### Группа 4: Аналитика и отчеты

**Правильный путь:** Аналитика → Производительность → Velocity

**Карточка 17:** "Burndown chart текущего спринта"  
**Правильный путь:** Аналитика → Производительность → Burndown chart

**Карточка 18:** "Отчет по завершенным задачам за месяц в формате PDF"  
**Правильный путь:** Аналитика → Отчеты → Экспорт отчетов

**Карточка 19:** "Производительность конкретного участника за квартал"  
**Правильный путь:** Аналитика → Отчеты → По участникам

**Карточка 20:** "Общий обзор статуса всех проектов на одном экране"  
**Правильный путь:** Аналитика → Обзор проектов

### Группа 5: Настройки и профиль

**Карточка 21:** "Изменить свой аватар и имя в профиле"  
**Правильный путь:** Настройки → Профиль → Личные данные

**Карточка 22:** "Настроить, чтобы не получать уведомления после 18:00"  
**Правильный путь:** Настройки → Уведомления → Рабочие часы

**Карточка 23:** "Включить двухфакторную аутентификацию для входа"  
**Правильный путь:** Настройки → Безопасность → Двухфакторная аутентификация

**Правильный путь:** Настройки → Интеграции → Slack

**Карточка 25:** "Посмотреть список всех активных сессий и выйти на других устройствах"  
**Правильный путь:** Настройки → Безопасность → Активные сессии

### Группа 6: Уведомления

**Карточка 26:** "Посмотреть все сообщения, где меня упомянули (@mention)"

**Правильный путь:** Уведомления → Непрочитанные

### Группа 7: Помощь и поддержка

**Карточка 29:** "Найти список всех горячих клавиш для быстрой работы"  
**Правильный путь:** Помощь → Горячие клавиши

**Карточка 30:** "Написать сообщение в службу поддержки о проблеме"  
**Правильный путь:** Помощь → Поддержка → Написать в поддержку

## Категории сложности карточек

### Легкие (85%+ правильных ответов ожидается):
- Карточки: 2, 3, 6, 7, 11, 12, 21, 28, 30
- **Причина:** Очевидное размещение, прямая терминология

### Средние (70-85% правильных):
- Карточки: 1, 4, 8, 9, 10, 13, 16, 17, 20, 22, 23, 26, 29
- **Причина:** Требуют понимания структуры, но логичны

### Сложные (50-70% правильных):
- Карточки: 5, 14, 15, 18, 19, 24, 25, 27
- **Причина:** Неоднозначное размещение, альтернативные пути

## Альтернативные правильные пути

**Карточка 27** (Уведомления) - 2 правильных пути:
1. Уведомления → Настройки уведомлений
2. Настройки → Уведомления → Email уведомления

**Карточка 12** (Пригласить) - может быть также:
- Настройки → Организация → Пользователи (для Admin)

**Карточка 3** (Создать задачу) - альтернативный путь:
- Проекты → [Выбрать проект] → Создать задачу

## Метрики успеха

**Целевые показатели:**
- Ключевые разделы (Проекты, Задачи, Команда): ≥ 85% попаданий
- Важные разделы (Аналитика, Настройки): ≥ 75% попаданий
- Второстепенные (Помощь): ≥ 65% попаданий

**Критерии проблемности раздела:**
- < 60% попаданий = критическая проблема
- 60-70% = требует улучшения
- 70-85% = хорошо, можно улучшить
- > 85% = отлично
