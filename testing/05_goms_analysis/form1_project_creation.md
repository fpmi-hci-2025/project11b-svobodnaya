# GOMS Анализ: Форма создания проекта
## ProjectFlow - Web приложение

---

## Описание формы

**Форма:** Создание нового проекта  
**Контекст:** Главная страница → Кнопка "+ Создать проект"  
**Цель:** Создать проект со всеми основными параметрами и командой

### Поля формы:

1. **Название проекта** (текстовое, обязательное)
2. **Описание** (textarea, обязательное)
3. **Методология** (dropdown: Scrum/Kanban/Waterfall)
4. **Длительность спринта** (dropdown: 1/2/3/4 недели) - только для Scrum
5. **Дата начала** (datepicker)
6. **Дата окончания** (datepicker)
7. **Видимость** (radio buttons: Публичный/Внутренний/Приватный)
8. **Рабочие дни** (checkbox group: Пн-Вс)
9. **Часовой пояс** (dropdown с поиском)
10. **Команда** (multi-select с поиском, добавление 5 участников)
11. Кнопки: **"Создать"** (primary) и **"Отмена"** (secondary)

**Общее количество полей:** 10 + действие с кнопкой = 11 действий

### Расчет по модели GOMS:

**Операторы:**
- H (Hand movement) = 0.4 с × 2 = 0.8 с
- P (Point) = 1.1 с × 1 = 1.1 с
- K (Keystroke) = 0.2 с × (35+60+10+10+5+5+11+14+12+13+15 + 23 прочих) = 0.2 × 203 = 40.6 с
- M (Mental) = 1.35 с × 5 = 6.75 с

**Общее время:** 0.8 + 1.1 + 40.6 + 6.75 = **49.25 секунд** ≈ **49 секунд**

**Эффективность:** Высокая, минимум переключений между устройствами

## Детальный анализ узких мест

### Проблемные зоны:

1. **Переключение рука ↔ мышь (2A, 2B):**
   - 15 переключений × 0.4с = 6 секунд потерь
   - **Рекомендация:** Добавить больше горячих клавиш

2. **Перемещение указателя мыши (2A, 2B):**
   - 23 перемещения × 1.1с = 25.3 секунд
   - **Рекомендация:** Крупные клик-таргеты, логичное расположение

3. **Ввод текста в поле "Команда" (все варианты):**
   - 65 символов для поиска 5 участников
   - **Рекомендация:** История, быстрый выбор, шаблоны команд

4. **Ментальные паузы (все варианты):**
   - 5 пауз × 1.35с = 6.75 секунд
   - **Рекомендация:** Подсказки, примеры, defaults

## Рекомендации по оптимизации формы

### Высокий приоритет:

1. ✅ **Реализовать полную Tab-навигацию**
   - Все поля доступны с клавиатуры
   - Enter для submit
   - Экономия: ~31 секунда (39%)

2. ✅ **Добавить template descriptions**
   - Типовые описания для проектов
   - Экономия: ~11 секунд (14%)

3. ✅ **Автоопределение часового пояса**
   - По IP или настройкам браузера
   - Экономия: ~1.2 секунды (2%)

### Средний приоритет:

4. ✅ **История команд / Шаблоны**
   - Сохранение последней команды
   - Быстрые команды (Frontend team, Backend team)
   - Экономия: ~8 секунд (10%)

5. ✅ **Интеллектуальные defaults**
   - Методология: последняя использованная
   - Рабочие дни: Пн-Пт по умолчанию
   - Видимость: из настроек организации

6. ✅ **Autocomplete для участников**
   - Предложения при вводе 2-3 символов
   - Сортировка по частоте взаимодействия

### Низкий приоритет:

7. ✅ **Горячие клавиши**
   - Ctrl+Enter для создания
   - Esc для отмены
   - Ctrl+1/2/3 для методологий

8. ✅ **Копирование из шаблона**
   - Создать проект на основе существующего
   - 1 клик вместо заполнения всей формы
